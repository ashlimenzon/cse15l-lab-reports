# Command: cd
**No argument**
```
shlim@Abi_HP_Laptop MINGW64
$ cd

```
**Path to directory as argument**
```
shlim@Abi_HP_Laptop MINGW64 ~
$ cd lecture1

shlim@Abi_HP_Laptop MINGW64 ~/lecture1 (main)
$
```
**Path to file as argument**
```
shlim@Abi_HP_Laptop MINGW64 ~/lecture1 (main)
$ cd Hello.java
bash: cd: Hello.java: Not a directory

```
